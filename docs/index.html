    Resumen:

    Este curso intensivo ofrece una introducción práctica a tres herramientas fundamentales para la ciencia de datos y el desarrollo reproducible: Git para control de versiones, Bash para automatización en la terminal y R para análisis estadístico y visualización. No se requiere experiencia previa: comenzaremos desde cero y avanzaremos paso a paso, combinando teoría y práctica en un entorno accesible.

    Enfoque de la audiencia:

    • Público general interesado en aprender herramientas digitales: no se requiere experiencia previa, el curso inicia desde cero.

    • Estudiantes y profesionales de cualquier disciplina que deseen mejorar sus habilidades en manejo de datos, control de versiones y automatización.

    Personas con interés en ciencia de datos, programación o investigación reproducible, pero que aún no han trabajado con Git, Bash o R.
